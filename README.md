# RedPill Gateway

A fast AI gateway for routing requests to 250+ LLMs.

## Quick Start

```bash
# Install dependencies
npm install

# Start the gateway
npm run dev:node
```

Gateway runs on `http://localhost:8787`

## Build

```bash
npm run build
node build/start-server.js
```
